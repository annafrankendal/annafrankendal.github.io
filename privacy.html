1. Personuppgifter vi samlar in

Om du kontaktar oss via e-post kan vi behandla de uppgifter du själv väljer att dela, till exempel namn, e-postadress och meddelandeinnehåll.

2. Varför vi samlar in uppgifter

Vi använder uppgifterna för att kunna svara på din kontaktförfrågan och hantera fortsatt dialog om samarbete eller uppdrag.

3. Rättslig grund

Behandlingen sker med stöd av berättigat intresse eller samtycke, beroende på hur kontakten initieras.

4. Lagring

Uppgifterna lagras endast så länge de behövs för att hantera kontakten och följa upp relevanta förfrågningar.

5. Hur länge vi sparar uppgifter

Uppgifter sparas så länge de behövs för återkoppling och uppföljning, och raderas därefter.

6. Dina rättigheter

Du kan begära utdrag, rättelse eller radering av dina uppgifter. Kontakta oss via e-post för att utöva dina rättigheter.
